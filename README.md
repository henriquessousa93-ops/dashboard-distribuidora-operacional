# dashboard-distribuidora-operacional
Dashboard desenvolvido em Power BI para análise operacional de uma distribuidora, incluindo pedidos, status de entrega, controle de estoque e indicadores de performance com dados fictícios.

![Dashboard](https://raw.githubusercontent.com/henriquessousa93-ops/dashboard-distribuidora-operacional/main/assets/dashboard.gif)

🎯 Objetivo

Simular um cenário de operação de uma distribuidora para análise de pedidos, entregas e estoque.

📌 Problema

A gestão de pedidos, entregas e estoque é essencial para garantir eficiência operacional e evitar perdas.

📈 Análises desenvolvidas

Evolução da quantidade de pedidos ao longo do tempo

Status de entrega (no prazo vs atrasado)

Controle de estoque de matéria-prima

Indicadores gerais:

Total de pedidos

Valor movimentado

Ticket médio

🛠️ Ferramentas utilizadas

Power BI

📊 Insights possíveis

Identificação de atrasos logísticos

Análise de desempenho de entregas

Melhor controle de estoque

⚠️ Observação

Dados fictícios utilizados para fins educacionais.
